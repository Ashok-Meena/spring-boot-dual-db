# Spring Boot Dual Database Project (MySQL + Azure SQL Server)

This project demonstrates how to configure and use **two separate databases** in a single Spring Boot application using **Spring Data JPA**.

## 🚀 Features

- Dual DB configuration (MySQL as Primary, PostgreSQL as Secondary)
- Separate `EntityManagerFactory` and `TransactionManager`
- Clean multi-database architecture
- REST API to demonstrate CRUD operations from both DBs
- Swagger UI integration (optional)

## 🛠 Tech Stack

- Java 17+
- Spring Boot 3.x
- Spring Data JPA
- MySQL
- PostgreSQL
- Maven
- Lombok
